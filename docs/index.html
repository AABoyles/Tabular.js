    Basic Usage

    To Use Tabular.js, just source the library however you prefer:

    <script src="tabular.js"></script>

    And then access the power of tabular.js by calling the global object it creates (creatively called "tabular"):

    console.log(tabular);

    Try putting your tab-delimited data in an element you can select using Javascript:

    var container = document.getElementById("csv");
var tabbedData = container.innerHTML;
var tableHTML = tabular.csv2Table(tabbedData);
container.innerHTML = tableHTML;

    If you use the jQuery plugin, you can cut that whole block to a single line:

    $("#csv").csv2Table();

    Functions

    table2Array(table)

    Given a table element, return a 2-dimensional array.

    table2ObjectArray(table)

    Given a table element, return an array of objects mapping the first row (presumably titles) to the given row's corresponding value.

    table2CSV(table)

    Given a table element, returns a CSV String.

    table2CSVURL(table)

    Given a table Element, return a URL-encoded version of the CSV string representation to allow downloading of the data direct from the browser (no server required).

    csv2Table(csv, delim, headers)

    Given a CSV string, a delimiter string, and a boolean indicating the presence of headers in the CSV string, return an HTML table string

    csv2Array(csv, delim)

    Given a CSV string and a delimiter string, return a 2 dimensional javascript array.

    csv2Object(csv, delim)

    Given a CSV string and a delimiter string, return a javascript object.

    jQuery Functions

    And here's what's currently available with the jQuery Plugin:

    csv2Table(delim, headers)

    Transforms the element upon which it was called into a table. Assumes it contains CSV-parsable content.

    This may seem a little silly at first, but the intent is to provide a seamless route from a textarea into which users paste CSV text to parsed, fully functional table. A Demonstration of this general principle is given in this example.

    table2CSV(delim)

    Returns the selected tables' CSV-encoded contents.

    If the selector refers to multiple elements, returns an array of CSV strings.

    downloadTable(format, send)

    Returns an array of URL-encoded CSV Strings for use in downloadable links

    If the selector refers to multiple elements, returns an array of CSV-url strings.
